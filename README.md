# Session Sage 🎵

A repertoire & set list manager for Irish bands (and any band, really).

## Features

- **Repertoire Management** — Track songs (vocals) and tunes (instrumentals) with key, tempo, and notes
- **Three statuses** — Mark pieces as *Known*, *Learning*, or *Wishlist*
- **Set List Builder** — Create set lists for gigs with date, venue, and ordering
- **Shared Data** — All band members see the same data via localStorage
- **Search & Filter** — Quickly find songs by name, status, or type

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS + shadcn/ui

## Development

```bash
pnpm install
pnpm dev
```

## Deploy on Vercel

1. Import this repo on [vercel.com](https://vercel.com)
2. Framework preset: **Vite**
3. Click **Deploy**
